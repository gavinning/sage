# 快速开始
***

### Installs
* [Install Nodejs](http://nodejs.org/)
* [Install UZ](https://www.npmjs.com/package/uz)
* [Install Aimee-cli](https://www.npmjs.com/package/aimee-cli)

### Create demo
```bash
$ aimee create pro
$ cd pro
$ uz release -p
$ uz server start
```

### Create page
```bash
$ aimee create -p home
```

### Create widget app
```bash
$ aimee create -w header
```
