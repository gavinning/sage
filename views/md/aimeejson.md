# Aimee.json
App的描述文件，用于描述App的相关信息
```js
{
  // 必选，描述App的名字
  "name": "sage",
  // 必选，描述App的版本
  "version": "1.0.0",
  // 可选，描述App的信息，推荐
  "description": "sage",
  // 可选，描述App的作者
  "author": "gavinning",
  // 可选，描述App的关键词，方便使用者搜索
  "keywords": "sage web dev",
  // 可选，描述App的依赖关系，有依赖则必选
  "dependencies": {
    "zepto": "~1.0.1"
  }
}
```
